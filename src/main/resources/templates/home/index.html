Bouncing Balls Project

By Andrea Di Claudio, Nicolas Javier Jan, Daniil Strelan

• Implementation

On the first meeting we started by reading the assignment and deciding
our initial scope.

From the very beginning we wanted to create a small software with a
single ball bouncing correctly on the border of the scene created in
JavaFX to better understand how to work with collisions.

After achieving a ball bouncing correctly on the scene border, we decided
to add a second ball with different size.
The new ball is "heavier" which means that the first (blue) ball will
bounce off when it collides with the second (purple) ball.

• The Code

The project has two java classes and a single FXML file.

One class is the controller class which contains the method to move the ball
(one method for each ball), the "initialize" method, and the JavaFX
properties.

Inside the "initialize" method we have created two threads, one for each ball.
The threads call the "moveBall" method.

The "moveBall" method uses the "TimeLine" class which is an animation
and processes individual keyframe sequentially.
Inside the method we have also collision detections both for the scene
and the balls.

The second class is the actual application that has a "start" method and
the "main" method.

The "start" method has an FXMLoader object that loads an object
hierarchy from an XML document.

The "main" method launches the application.
